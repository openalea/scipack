Welcome to Openalea Scipack's documentation!
=====================================

.. figure:: _static/fromt_image.png

 **Example of package explorer entries of Visualea.**

Openalea Scipack is a set of python wrappers that brings several capabilities to `Visualea <https://visualea.readthedocs.io>`_
the OpenAlea Visual Programming environment:

- image manipulation, the possibilities range from image loading/saving to morphology operations or registration.
- some `Numpy <https://numpy.org/>`_ functionalities like trigonometric and hyperbolic functions, array manipulation, etc.
- plot functionalities from `matplotlib/pylab <https://matplotlib.org/stable/api/pylab.html>`_.
- standard logical components based on python libraries.
